# PHPの最初の最初
## 雑多のこと
まずは、文法などの解説の前に、少し予備知識を付けておきましょう。
PHP(Hypertext Preprocessor)は1995年に産声を上げて、1998年にはPHP 3.0が公開され、ちょうどインターネットの普及に伴い、多くのプログラマーに利用されるようになりました。
2013年現在、PHP 5.4がリリースされています。
## インストール
### Windows
PHPのダウンロードサイト、http://windows.php.net/download/#php-5.3 に行って、インストーラー付きをダウンロードして実行すれば簡単にインストールできます。
ただし、ＰＨＰタンタンでもコマンドラインでコードを実行することが出来ますが、ＷＥＢアプリケーションとしてのＰＨＰは単体では動きません。そこでApache(Httpサーバー)をインストールして、PHPが動くようにすることも可能ですが、XAMPPを利用した方が簡単です。
XAMPPはApacheやPHPはもとより、MySQLやFileZillaなども一気にインストールできちゃいます。ただ、コマンドラインなどから実行できるよう、PHPにパスを通しておきましょう。例えば、c:\xampp\phpにインストールされているとしたら、「コントロールパネル→システム→システムの詳細→環境変数設定」で下のPathというのをダブルクリックして「;c:\xampp\php」を最後に書き加えて下さい。
![環境変数設定](path.png)
## コーディングスタイル
* PEAR標準コーディング規約
* WordPress コーディング基準
* Zend Framework PHP 標準コーディング規約
など、PHPには色々とコーディング規約があり、しかも言っていることが相当違います。例えば、Zendでは変数名などではキャメルスタイル(userProfileなど)を使いなさいと言い、WordPressではアンダースコア(user_profile)を使え、など真っ向から対立している場合もあります(その他に、タブを使えとか、スペースを使えとか色々違います)。
すなわち、結局の所、この辺の規約を読んで、好きなようにすれば良い、ということになりそうです。ただ首尾一貫しないと読みづらいものになるので、好きなものを見つけて、自分なりのスタイルを構築しましょう!
## スクリプト言語
PHPは、CやC++と違い、実行するためにコンパイル(機械語への変換作業)を行う必要がないスクリプト言語です。
  
```php
<?php
    echo "Hello World, baby!"
?>
```
  
上記のような内容(行番号は除く)のファイル(hello.php)を用意し、Windowsであれば「スタートボタン→アクセサリー→コマンドプロンプト」でDos窓を開き、PHPにパスが通っていれば、
```bash
php hello.php
```
を入力して、実行すれば、「Hello World, baby!」が表示されます。
